# 🛒 MyStore – Web Application

A modern and interactive **e-commerce web application** built using HTML, CSS, and JavaScript.
Designed with a premium UI, smooth animations, and a user-friendly experience.

---

## 🚀 Live Demo

🌐 https://bugbyte-code.github.io/web-app/

---

## ✨ Features

* 🔐 User Registration & Login
* 👤 Personalized User Profile with Animated Avatar
* 🛍️ Product Catalog with Images & Pricing
* 🛒 Add to Cart Functionality
* 🌙 Dark Theme UI
* 🎭 Dynamic Cartoon Avatars (unique for each user)
* ✨ Smooth Animations & Modern Design

---

## 🧑‍💻 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Animations
* **JavaScript** – Functionality
* **LocalStorage** – Data handling
